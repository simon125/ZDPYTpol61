CSS jest potężny

Za pomocą CSSa możemy

  • stylować czcionkę, kolorować, kontrolować rozmiar, spacing między znakami itd
  • korować tło, ramki i praktycznie wszystkie inne rzeczy w htmlu
  • pozycjonować - ustawiać elementy HTML tak jak to określimy
  • chować, pokazywać, powiększać elementy
  • zmieniać "naturę" elementów - blokowość vs liniowość
  • animować elementy
  • wiele wiele innych rzeczy

żeby to wszystko osiągnąć trzeba jedynie użyć odpowiednich propertisów i wartości
